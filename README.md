# 🏨 Hotel Njenka Website

A modern and responsive hotel website designed to showcase hotel services, rooms, and contact information. The **Hotel Njenka Website** provides visitors with an easy way to explore hotel offerings and learn more about available accommodations.

---

## 📖 Project Overview

The **Hotel Njenka Website** is a front-end web development project created to simulate a real-world hotel website. The goal of this project was to design a clean, responsive, and user-friendly interface that displays essential hotel information such as rooms, services, and contact details.

This project demonstrates skills in **HTML, CSS, and JavaScript**, along with responsive web design techniques.
